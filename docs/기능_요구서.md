# devops-assist 기능 요구서

> 버전: 0.1 | 작성일: 2026-02-27 | 상태: 초안

## 1. 개요

**devops-assist**는 개인 지식 관리 웹 서비스다.
사용자는 계정을 생성하고 자신만의 워크스페이스에서 문서·노트를 작성하며, AI가 관련 문서를 추천하고 요약·보완을 도와준다.

### 개발 단계

| Phase | 내용 | 상태 |
|-------|------|------|
| **A** | 문서/노트 중심 + AI 보조 + 계정 기반 개인 워크스페이스 | 현재 |
| B | 로컬/GitHub 프로젝트 연결 → AI 분석·지식화 | 예정 |
| C | 혼합 (문서 + 파일 + 링크 통합) | 예정 |

## 2. 사용자 관리

### 2.1 인증

| ID | 요구사항 |
|----|---------|
| AUTH-01 | 이메일 + 비밀번호로 회원가입 |

| AUTH-04 | 로그아웃 시 Refresh Token 무효화 |

| ID | 요구사항 |
|----|---------|

| WS-02 | 워크스페이스는 폴더 트리 + 문서 목록으로 구성 |

## 3. 문서 에디터

### 3.1 에디터 형태

블록 기반 리치 텍스트 에디터 (Notion 스타일)

| ID | 요구사항 |
|----|---------|
| ED-01 | `/` 입력으로 블록 타입 선택 (slash command) |
| ED-02 | 지원 블록: 단락, 제목(H1~H3), 코드블록, 목록(순서/비순서), 인용, 구분선 |
| ED-03 | 코드블록은 언어 지정 + 신택스 하이라이팅 |
| ED-04 | 인라인 서식: **굵게**, *기울임*, `코드`, ~~취소선~~, 링크 |
| ED-05 | 블록 드래그 앤 드롭으로 순서 변경 |
| ED-06 | 자동 저장 (마지막 편집 후 3초) |
| ED-07 | 문서 제목은 별도 입력란 (에디터 영역 위) |

### 3.2 문서 메타데이터

| ID | 요구사항 |
|----|---------|
| META-01 | 생성일시, 수정일시 자동 기록 |
| META-02 | 태그 다중 입력 (입력 후 Enter/쉼표로 추가) |
| META-03 | 문서별 즐겨찾기 표시 |

## 5. AI 기능

> AI는 **보조 역할**이다. 사용자가 주도하고, AI는 제안·추천·요약으로 돕는다.
> 인라인 제안(Tab 수락)과 온디맨드 기능(버튼 클릭)으로 구분한다.

### 5.1 인라인 제안 — Tab-to-Accept (1순위)

사용자가 입력창에 포커스하거나 비워두면 AI가 고스트 텍스트로 제안을 표시한다.
Tab 키를 누르면 제안이 입력창에 자동 입력된다.

| ID | 입력 위치 | 동작 |
|----|-----------|------|
| AI-01 | 태그 입력창 | 문서 내용 기반으로 태그 후보 표시. Tab → 첫 번째 태그 수락. 계속 Tab → 다음 태그 추가 |
| AI-02 | 폴더명 입력창 | 새 폴더 생성 시 문서 맥락 기반으로 폴더명 제안 |
| AI-03 | 문서 제목 입력창 | 제목이 비어있거나 "제목 없음"일 때 내용 기반으로 제목 제안 |
| AI-04 | 제안 취소 | Esc 키 또는 다른 입력 시 제안 해제 |
| AI-05 | 응답 속도 | 포커스 후 1초 이내 제안 표시 (사용자 체감 지연 최소화) |

### 5.2 관련 문서 추천 (1순위)

| ID | 요구사항 |
|----|---------|
| AI-06 | 문서 열람 시 사이드바에 유사 문서 최대 5개 추천 |
| AI-07 | 추천 기준: 내용 유사도 + 공통 태그 |
| AI-08 | 추천 문서 클릭 시 해당 문서로 이동 |

### 5.3 요약 및 보완 제안 (1순위)

| ID | 요구사항 |
|----|---------|
| AI-09 | 문서 요약 버튼 → 현재 문서 3~5줄 요약 생성 |
| AI-10 | 보완 제안 버튼 → "이런 내용을 추가하면 좋겠어요" 형태로 제안 |
| AI-11 | AI 응답은 스트리밍으로 표시 (SSE) |
| AI-12 | 요약/제안 결과를 문서에 직접 삽입하는 버튼 제공 |

### 5.4 자연어 검색 (2순위)

| ID | 요구사항 |
|----|---------|
| AI-13 | 검색창에 자연어 질의 입력 |
| AI-14 | 워크스페이스 내 문서 전체에서 의미 기반 검색 |
| AI-15 | 검색 결과에 관련 문장 하이라이팅 |
